# Quiziest

A Programming quiz site, where you can test your JavaScript, React, CSS, and Git skills.

## Live Link

Hosted in Netlify -> [Quiziest](https://quiziest.netlify.app/)

## Features

- View correct answer options
- Statistics Of Total Quizzes
- Easy navigation
- Responsive
